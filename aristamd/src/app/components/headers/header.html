CSS
.expandcollapse-heading-collapsed span{ font-size: 16px; margin:0px; } .expandcollapse-heading-expanded span{ font-size: 32px; } .expandcollapse-heading-collapsed{ height:20px; } .expandcollapse-heading-collapsed:not(.noHover):hover{ padding-bottom: 30px; } .animate-show-hide { position: absolute; transition:0.2s opacity linear; transition-delay: 0.3s; opacity:1; top:73px; } .animate-show-hide.ng-hide { position: absolute; transition:0.2s opacity linear; transition-delay: 0.1s; opacity:1; }
Options
isExpanded: boolean If the header is expanded as default. Default: false titleLeft: string Title on the top left contentLeft: string (template is acceptable) Content that is on the left of the header titleRight: string Title on the top right contentRight: string (template is acceptable) Content that is on the right of the header nudge: boolean Property that enable the on mouseover and on mouseout animations. Default: true transitionTime: number (300 ms default) Transition time for the animations app-header-directive : directive (required) Directive Required to apply the animations
